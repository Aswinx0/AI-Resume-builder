## 🧠 AI Resume Builder

The AI Resume Builder is an intelligent web application that helps users create professional, customized resumes effortlessly. Powered by Generative AI, it automatically analyzes user inputs and generates optimized resume content — including summaries, skill descriptions, and experience highlights — tailored to specific job roles or industries.

Built using React.js and Vite, the app provides an intuitive interface with live preview, multiple templates, and section-based editing (Personal Info, Education, Skills, Projects, etc.). It leverages Gemini API (Google Generative AI) for natural language generation and Firebase for authentication and data storage.

## 🚀 Key Features:

AI-powered resume content generation

Real-time resume preview

Editable sections (Personal Info, Skills, Education, Experience, Projects)

Customizable templates and themes

Responsive and user-friendly interface

## 🛠️ Tech Stack:
React.js | Vite | Tailwind CSS  | Gemini API
